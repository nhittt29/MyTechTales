Mình viết về Java (socket TCP, đa tuyến, quản lý I/O, dịch vụ mạng) và JavaScript (API, Node.js, WebSocket, JSON).

Bạn sẽ khám phá cách các hệ thống trao đổi dữ liệu, giao tiếp và vận hành hiệu quả.

  • Lập trình Java cơ bản và đa tuyến
  • Quản lý socket TCP và luồng I/O
  • Dịch vụ kết nối mạng với Java
